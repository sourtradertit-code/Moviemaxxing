# botMovie

A Telegram bot for movie-related functionality built with Python and aiogram 3.

## Setup

1. Set the `BOT_TOKEN` environment variable with your Telegram Bot API token (get one from [@BotFather](https://t.me/BotFather)).
2. Run the bot using the **Start application** workflow.

## Tech Stack

- **Language:** Python 3.11
- **Framework:** aiogram 3.13.0 (async Telegram Bot API)
- **Database:** aiosqlite (SQLite, async)
- **Utilities:** thefuzz (fuzzy string matching), requests
